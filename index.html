hexo 搭建

hexo 搭建详细步骤

  • 安装必要工具：
    sudo apt-get install nodejs
    sudo apt-get install npm
    • 如果 node不是最新，用下面的方法
      sudo npm install -g n
      node -v
      sudo n latest
      PATH=”$PATH”
      node -v
  • 安装hexo
    sudo npm install -g hexo-cli
  • 创建hexo工程
    mkdir -p hexo_blog
    cd hexo_blog
  • 在hexo_blog目录下创建一个文件夹blog，使用Hexo命令初始化blog为hexo工程目录。
    hexo init blog
  • 新建POST，blog/source/_posts/HelloWorld.md
    hexo new “HelloWorld”
  • 生成静态文件,使用Hexo引擎将Markdown格式的文件解析成可以使用浏览器查看的HTML文件，HTML文件存储在blog/public目录下
    hexo generate
  • 运行hexo服务器，-p指定其他端口
    hexo server
    hexo server -p 4001
  • 修改 blog/sources/_post/HelloWorld.md， 刷新浏览器即可预览

修改导航菜单：添加分类/标签

  • 修改配置文件

  • 创建对应页面

    • home 和 archives 默认都有
    • 创建categories
      • hexo new page categories
      • vi ~/blog/source/categories/index.md 添加一行 type: categories
    • 创建tags
      • hexo new page tags
      • vi ~/blog/source/tags/index.md 添加一行 type: tags

  • 修改对应文件.md加入 tags/categories

  • 重新布局就可以看到效果啦
